## [0.1.0] - 2024-XX-XX

### Added
- Initial release of GoVM
- Install Go versions from official Go downloads
- Switch between installed Go versions
- List installed and remote available Go versions
- Uninstall Go versions
- Import system-installed Go
- Detect and warn about PATH conflicts
- Cross-platform support (Windows, Linux, macOS)
- Multiple architecture support (x86_64, ARM64, ARMv7)
- Shell install scripts for easy installation
- GitHub Actions workflow for automated releases
